Section 3: The Build Process

Follow this five-step process to get your local AI running. Each step builds upon the last, from initial installation to your final knowledge-base query.

Step 1 & 2: Install the Toolchain

Use the Pinokio browser to find and install Ollama first, then Open WebUI. Launch both and ensure you can access them at `http://localhost:11434` (for Ollama) and `http://localhost:3000` (for Open WebUI).

Step 3: Pull a Model

You need a model to chat with. Use the command line to pull one via Ollama. Llama 3 8B is recommended, but smaller alternatives like Phi-3 are great for systems with less RAM. This chart shows their approximate download sizes.
